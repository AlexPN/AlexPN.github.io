JS/CSS Simple Code Block Styling and Syntax Highlight

During my process of building this site, I needed syntax highlighting for code blocks. Although there are many available syntax highlighters online (e.g. highlight.js), I decided instead to make a very lightweight version. As usual, there is something to learn, but the goal is to convert this default, ugly code HTML code block: 

	
#include 
#include 

//program entry
void main(){


	char c = 'y';
	char * s = "hello";
	int ten = 10;
	/*using, printf,
	print the above*/

	printf("out: %c %s %d\n",c,s,ten);
}

To something much more readable: 

#include 
#include 

//program entry
void main(){


	char c = 'y';
	char * s = "hello";
	int ten = 10;
	/*using, printf,
	print the above*/

	printf("out: %c %s %d\n",c,s,ten);
}

The Styling

We would like to add the following to the element in general:

  • Visual separator from other content. In my case, I chose a different color for the block and a drop shadow.

  • Line numbers.

  • Monospaced font.

  • No word-wrap, but horizontal scrolling with a dynamic scroll-bar.

Creating the Visual Separation

We will start with a CSS template that will change thhe appearance of <pre> tags that are part of a named class. To start, just change the background color of <pre> wrapped data and change any text that is within a <code> tag within a <pre> tag. 

	
pre.code {
        display: block;
	background: #282b2e;
}

pre.code code {
	color: #bccada;
}

It still looks blandish... so, let's add some padding to the inner text and a drop shadow. Also, let's make the code font monospaced and easier to read. 

	
pre.code {
        display: block;
	background: #282b2e;
        padding: 1em 5em 1em 2em;
        box-shadow: 0px 0px 4px 4px rgba(0,0,0,0.2);
}

pre.code code {
	color: #bccada;
	font: normal 10pt Consolas, monospace;
}
The Horizontal Scroll Bar

Now you can notice that if you resize your window that the inner text just spills out of the block. We need to add a horizontal scroll bar that only appears when needed. Luckily, there is a CSS parameter called overflow-x that can be set to auto to make this happen. 

	
pre.code {
        display: block;
	background: #282b2e;
        padding: 1em 5em 1em 2em;
        box-shadow: 0px 0px 4px 4px rgba(0,0,0,0.2);
	overflow-x: auto;
}

pre.code code {
	color: #bccada;
	font: normal 10pt Consolas, monospace;
}
Line Numbers

Adding line numbers to the block is trickier as we need to make use of counters. The way a counter works is that you set an identifier on which the counter will reset (e.g. <pre>), an identifier on which to increment (e.g. <code>), and an identifier on which to display. As well, wee will need to use :before commands in CSS to display the line numbers before each element.
